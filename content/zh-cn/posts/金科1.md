## 3. 三元運算符 

    value = condition and a or b
    //當a 等於 0 或者 是 false 的時候 value 不会赋值成 0 或 false
    //要改成 
    value = condition ? a : b

    //类似的
    value = a or b
    //改成
    value = a != undefined ? a : b
